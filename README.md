# Escondido Pet Care Website

This is a simple static site ready for GitHub + Vercel.

## Important
Before publishing, open `index.html` and replace:

`YOUR-EMAIL-HERE`

with the email address where you want form submissions sent.

## Deploy steps
1. Upload all files from this folder into your GitHub repo.
2. Import the repo into Vercel.
3. Deploy.
4. Add your custom domain in Vercel: `escondidopetcare.com`
5. Point your registrar DNS to Vercel when prompted.
